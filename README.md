# Restaurant Landing Page

## About

A responsive Italian restaurant landing page showcasing delicious food and restaurant services. Built with vanilla HTML, CSS, and JavaScript, featuring a modern navigation bar, hero section, and interactive elements. The site provides information about the restaurant with an elegant design focused on food presentation and user engagement.

## Built With

- HTML5
- CSS3
- JavaScript
- FontAwesome Icons
- Responsive Design

## Getting Started

### Prerequisites

No installation required. Simply open the project in a web browser.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tumansutradhar/restaurant-landing-page
   cd restaurant-landing-page
   ```

2. Open the project:
   ```bash
   # Open index.html in your browser
   # Or use a local server
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

## Usage

- Navigate through different sections using the navigation bar
- Click on menu items to explore different sections
- View food images and restaurant information
- Contact the restaurant through the provided contact section
